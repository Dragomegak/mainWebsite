Desktop Environmental Setup Manager

This is a Microsoft Windows Program application and uses a QT Frontend with C++ and Win32API backend to create a virtual desktop based on a profile system. The profiles are made ahead of time and have a list of programs to run. The profiles can be made for different scenarios that require different programs to be launched.

Technologies Used

  • C++
  • QT Framework
  • Win32 API
